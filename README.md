
# SKN Artefakty — Strona WWW

Krótki projekt frontendowy zbudowany przy użyciu Vite + React + TypeScript.

- **Stos technologiczny:** React 19, TypeScript, Vite
- **Skrypty:** `npm run dev` (development), `npm run build` (produkcja), `npm run preview` (podgląd builda), `npm run lint` (sprawdzenie ESLint)

Szybkie uruchomienie:

1. Zainstaluj zależności:

```
npm install
```

2. Uruchom w trybie deweloperskim:

```
npm run dev
```

3. Zbuduj aplikację na produkcję:

```
npm run build
```

Struktura projektu (krótko):
- `src/` — kod źródłowy (komponenty, strony, style, dane)
- `public/` — zasoby statyczne
